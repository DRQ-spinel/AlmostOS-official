Quick start

配布は別サイトでやる前提。ここは“公式感”の表紙。

1) /assets の中身を好きに差し替える
2) index.html の “Get AlmostOS” リンクを配布サイトURLに変更
3) Render等の静的ホスティングにデプロイ（Publish directory: プロジェクトルート）

Routing

静的サイトなので基本は .html ページ。
SPA運用したいならリライトルールをホスト側で設定。

Link swap

配布URLは index.html のコメント箇所を差し替えるだけでOK。
