概要

これは何？

AlmostOSは、ブラウザ上で「OSっぽい体験」を再現するUIプロジェクト。 ドラッグできるウィンドウ、スタートメニュー、ターミナル、タスクマネージャなどを雰囲気重視で作ってます。

Desktop Windows Terminal TaskMgr BSOD

安全・プライバシー

  • 基本はローカルで完結（設定/配置はlocalStorageに保存）
  • 勝手にファイルを読んだり削除したりはしません
  • 内蔵Browserは外部サイトをiframeで開くので、通信が発生します

※「怖そうに見える演出」はあります（BSODなど）。

起動・操作ガイド

起動フロー

  1. boot.html を開く
  2. 必要なら F2/Del でBIOSへ
  3. F10 Save&Exit → 自動でOSへ

基本操作

  • タスクバーのアイコン → 起動/切替
  • ウィンドウの× → 終了
  • 右クリック → コンテキストメニュー
  • ダブルクリック → 最大化（対応ウィンドウ）

ショートカット

  • Ctrl + Space：Start
  • Ctrl + `：Terminal
  • Ctrl + Shift + Esc：Task Manager
  • Ctrl + L：Lock

Terminalは ↑/↓で履歴、Tabで補完。

タスクマネージャの仕様

アプリプロセスは停止できます。
システムプロセスを止めると即BSOD（演出）です。

機能

UI / 体験

  • デスクトップ / スタートメニュー / タスクバー
  • ウィンドウ操作（ドラッグ、前面化、リサイズ、最大化）
  • 右クリックメニュー、クイックパネル（設定トグル）
  • アイコン配置を保存（ドラッグ、複数選択、整列、スナップ）

システムっぽい機能

  • Terminal（コマンド、履歴、補完）
  • Task Manager（プロセス停止・簡易グラフ）
  • Lock Screen（PIN設定はlocalStorage）
  • HTML書き出し（状態入りも可）
  • BSOD（Stop Code表示＆再起動演出）

「入ってないもの」

  • 本当のOS機能（ドライバ、権限、実ディスク操作）
  • 勝手なファイル破壊
  • あなたのPCへの直接アクセス
  • 危険な挙動（マルウェア的なやつ）

内蔵アプリ / ゲーム

Stop Code（雰囲気説明）

クリックでコピーできます。説明は雰囲気なので、深追いはしないでね。

AlmostOS特有のやつ

タスクマネージャからシステムプロセスを停止 → 即BSOD（演出）。 「押しちゃダメ」を押すための装置です。

kernelish.exe sessionhost.exe taskbarhost.exe almostinit.exe windowdwm.exe

FAQ

壊れませんか？

壊れません。ブラウザ上のUIで、危険な操作は無効化しています。

BSODが出たんだけど大丈夫？

大丈夫。演出です。数秒で「再起動」して戻ります。

保存はどこに入りますか？

基本はlocalStorageです（端末/ブラウザごと）。HTML書き出しもできます。

Browserは何をしてる？

iframeで外部サイトを表示します。表示できないサイトは「外部」ボタンで新規タブへ。
